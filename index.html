WEB DEVELOPER

ALL ABOUT ME AND THE WORK I DO

CODING

Clean and valid!

I am a web developer who cares about correctly formatted HTML and tidy, well written CSS style sheets. I have been working in the industry for over 10 years and am familiar with Drupal, ASP.NET, and WordPress.

During my time working for a healthcare company I was asked to create clean HTML which would validate against W3C standards and conform with the WCAG 2.0 accessibility guidelines at level AA. This helps with cross-browser, cross-platform and future compatibility ensuring your site reaches it's maximum possible audience.

Thanks to my experience with multiple development environments I am able to code in various languages. When writing script for the client side I use JavaScript or jQuery. JavaScript libraries and frameworks come and go but I try to keep up to date with the with most popular. As for server side programming, I have used PHP, VB and C# in the past.

STYLING

CSS is my speciality

After years of dealing with browser inconsistencies and evolving design trends I consider myself to be a CSS expert. Unlike many modern developers I am able to create the layout of a website using divs and floats alone. Currently however, I tend to start with Bootstrap which uses media queries and responsive breakpoints to make websites look good on any device.

When appropriate I use the latest CSS3 techniques like subtle animation effects, Flexbox for navigation and grids for layout. Typography is also important to me and I am able to choose good looking web fonts whilst paying attention to readability and vertical rhythm.

When authoring CSS or SCSS using SASS I always ensure that my code is well structured and easy to read. This helps both myself and co-workers find relevant sections quickly and easily.
